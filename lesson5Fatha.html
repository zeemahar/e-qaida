Lesson No. 05

Fathath (First Type of Short Vowels)

(Page 05)

    Harakaath is the plural of Harakat means Movements. There are 6 following Movements in Arabic


  1. ( ـَـ ) Fatha
  2. ( ـِـ ) Kasra
  3. ( ـُـ ) Dhamma
  4. ( ـًـ ) Fathatan
  5. ( ـٍـ ) Kasratan
  6. ( ـٌـ ) Dhammatan
  1. Fatha
    1. The first Movement is Fathath which is always placed on the letter. While reading Fathath, Its pronunciation should not be prolonged, not to be jerked.
    2. The letter Alif will be considered as Hamzath if there is any Arabic symbol on the letter or down the letter.
    3. We should analyze all the letters,they have resemblance with each other,considering in Makhraj, such as ذ، ز، ص، ق، ك etc.
    4. If there is Fatha on the last letter, it will be pronounced with Sakoon while, stopping on the last letter.
    5. If there is round Taa in the end of word, it will be pronounced with Haa Sakinath instead of Taa while, stopping on the last letter.
