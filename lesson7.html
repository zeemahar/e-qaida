Tashdeed

  1. Stressing the letter is called Tashdeed.
  2. This sign looks like sign of (W).
  3. The letter having this symbol is called Mushaddad.
  4. Such letter is pronounced twice.
  5. Such letter has to be joined with the previous letter.
  6. If we have this symbol on the Letter Noon and Meem then we have to make nasal sound and will stay for one Alif, such rule is called Ghunna.
  7. All the stopping rules of the last letter will be implemented according to symbols (Movements) as we mentioned before in the end of each lesson while stopping on the last letter having Tashdeed.
